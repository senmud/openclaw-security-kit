# Xiaoxue Aegis (小雪盾)

## 核心功能
- **语义沙箱 (#765)**: 隔离外部输入，防止间接提示注入。
- **意图审计 (#652)**: 实时监测意图张力，拦截偏离目标的危险动作。
- **敏感保护 (#813)**: 物理级拦截对 API Key、.env 等文件的非授权读取。

## 安装
`clawhub install xiaoxue-aegis`
